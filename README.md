Ruby on Rails
-------------

This application requires:

- Ruby 2.3.3
- Rails 5.1.6

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).

Getting Started
---------------
``Run bundle install``

Start server with command:
``rails server or rails s``
